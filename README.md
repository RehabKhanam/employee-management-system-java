# Employee Management System

## Project Overview

This project is a simple Employee Management System developed using Java and Object-Oriented Programming (OOP) concepts. It stores and displays employee details such as Employee ID, Name, and Salary.

### Features

* Create Employee Objects
* Store Employee Information
* Display Employee Details
* Demonstrates OOP Concepts

### Concepts Used

* Classes and Objects
* Constructors
* Methods
* Encapsulation Basics

### Technologies Used

* Java
